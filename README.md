
# decision-tree-tutorial

Tutorial ini membahas tentang decision tree.

## Apa itu Decision Tree?

Decision tree (pohon keputusan) adalah salah satu algoritma machine learning yang digunakan untuk tugas klasifikasi dan regresi. Decision tree bekerja dengan memetakan setiap kemungkinan hasil dari serangkaian keputusan berdasarkan fitur-fitur data. Struktur pohon terdiri dari:

- **Root Node**: Node awal yang mewakili seluruh dataset.
- **Internal Node**: Node yang mewakili pengujian pada suatu fitur.
- **Leaf Node**: Node akhir yang mewakili keputusan atau hasil klasifikasi.

Keunggulan decision tree adalah mudah dipahami, divisualisasikan, dan tidak memerlukan banyak pra-pemrosesan data. Namun, decision tree rentan terhadap overfitting jika tidak dilakukan pemangkasan (pruning).

## Kegunaan Decision Tree

- Klasifikasi data
- Prediksi nilai (regresi)
- Analisis keputusan bisnis

## Contoh Sederhana

Misal, untuk menentukan apakah seseorang akan membeli komputer berdasarkan usia dan penghasilan, decision tree akan membagi data berdasarkan fitur-fitur tersebut hingga mencapai keputusan.
